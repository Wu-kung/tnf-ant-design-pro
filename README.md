# TNF Ant Design Pro

TNF fork of Ant Design Pro.

## Getting Started

Locally, you can run the following commands to start the development server:

```bash
$ pnpm install
$ pnpm dev
```

To build the production version, run:

```bash
$ pnpm build
```

To preview the production version, run:

```bash
$ pnpm preview
```
